# DevPack
Development tools for TomShelbyBot

## Commands
```
/disablepack
- Выключить пак

/enablepack
- Включить пак

/fatherexport
- Вывести информацию о всех командах в боте для экспорта в BotFather

/listpacks
- Показать список всех установленных паков

/metadel
- Удалить мету для чата по ключу

/metaget
- Получить из меты значение для чата по ключу

/metamap
- Показать карту меты для чата

/metaput
- Установить мету: <ключ> - <значение>

/saveall
- Сохранить мету
```
